# Language-Korrect
카카오톡 대화 내 욕설 분석 및 야민정음 판독기

## 사용
카카오톡 대화 내역 내보내기로 추출된 .txt 파일을 업로드합니다.

### 데이터
욕설데이터는 KR_LOL_CLIENT로부터 발췌된 데이터를 사용했습니다.  
야민정음 데이터는 나무위키를 참고하였습니다.
